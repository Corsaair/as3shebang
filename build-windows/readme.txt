Under Windows

you would want

1. install cygwin

2. copy as3sheabng.exe to /usr/bin

if you installed cygwin in C:\cygwin
that would be C:\cygwin\bin

3. copy also sheabng.bat into /usr/bin

4. be sure to add C:\cygwin\bin
   to your PATH env variable

5. install shebang as administrator

6. from ther you should be able to create and run
   AS3 shell scripts from a cygwin console
   and only from a cygwin console
